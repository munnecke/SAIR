# anomaly detection

Identifying data points that deviate significantly from expected patterns without using labeled examples. Applied at the [[LHC]] trigger level by [[Javier Duarte]] using autoencoders, allowing CMS and ATLAS to flag events that don't resemble the Standard Model — searching for new physics without specifying what signal to seek.
