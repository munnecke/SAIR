# reinforcement learning

A machine learning paradigm where an agent learns by interacting with an environment and receiving reward signals. Applied to [[LIGO]] servo control by [[RA Adhikari]] and to mathematical conjecture by [[Sergei Gukov]]. The [[sparse reward]] / long horizon problem is identified as the key unsolved challenge for RL in scientific discovery.
