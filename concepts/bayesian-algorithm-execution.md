# Bayesian Algorithm Execution

A framework introduced by [[Daniel Ratner]] (SLAC) that generalizes Bayesian optimization by decoupling the surrogate model from any fixed objective function. Allows the surrogate to learn underlying physics, making it applicable to arbitrary experimental queries. Extended in "[[Foundation BAX]]" to use large foundation models as surrogates.
