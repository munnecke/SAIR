# Montezuma's Revenge

An Atari video game notorious for requiring long action sequences before any reward is received — a canonical benchmark for [[sparse reward]] RL. [[Sergei Gukov]] uses it as an analogy: the RL community's solution ([[Go-Explore]]) came from diagnosing two specific failure modes (detachment and derailment) across 50-100 algorithm variants, rather than a single breakthrough.
